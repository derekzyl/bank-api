GET /api/v1/banks

Global Banks Index

Fetches all supported banks globally. The response is enveloped in a pagination container supporting page navigation and records limits.

QUERY PARAMETERS

PARAM TYPE DEFAULT
page int 1
limit int 50
country string -
REQUEST EXAMPLE 
curl -X GET "https://banks.cybergenii.com/api/v1/banks?page=1&limit=10"
JSON RESPONSE 
{
  "total": 442,
  "page": 1,
  "limit": 10,
  "data": [
    {
      "name": "Access Bank",
      "slug": "access-bank",
      "code": "044",
      "ussd": "*901#",
      "country": "Nigeria"
    }
  ]
}
GET /api/v1/nig/banks

Country Specific (NIG)

Retrieves the full, unpaginated dataset of all financial institutions in Nigeria. Use this if you require the complete list immediately without querying page offsets.

info

This endpoint is optimized for local caching and offline processing of regional bank lists.

REQUEST EXAMPLE 
curl -X GET "https://banks.cybergenii.com/api/v1/nig/banks"
GET /api/v1/nig/banks/[slug]

Single Bank Lookup

Retrieves metadata for a specific bank slug in the specified country namespace (e.g. Nigeria). Ideal for detailed account verification screens.

REQUEST EXAMPLE 
curl -X GET "https://banks.cybergenii.com/api/v1/nig/banks/access-bank"
GET /api/v1/nig/banks/[slug]/logo

Bank Logo CDN

Directly serves binary PNG/SVG images with long-term caching headers. Perfect for high-performance frontend implementation.
